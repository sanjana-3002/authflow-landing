# AuthFlow — Prior Authorization in 30 Seconds

AI-powered prior authorization for medical practices. Turn clinical notes and insurance cards into submission-ready PA forms in under 30 seconds.

**Live:** [authflow-landing.vercel.app](https://authflow-landing.vercel.app)

## Tech Stack

- **React + TypeScript** — component-based UI
- **Vite** — fast dev server and build tooling
- **Tailwind CSS** — utility-first styling
- **framer-motion** — scroll animations, parallax, and micro-interactions
- **Three.js** — WebGL particle field background
- **lucide-react** — icons

## Getting Started

```sh
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Deploy

```sh
npx vercel deploy --prod
```
